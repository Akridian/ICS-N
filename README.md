# Учебный репозиторий
Для курсовых работ по Распределенным системам ИСТбд31 (весна 2015 год)
# Правила работы
1. Курсовую работу запаковать в архив формата zip в названии архива указать предмет, год и ФИО авторов (Пример: ИКСиС_2017_КР_ВасилийПП_БобриковИП_'Название Работы'.zip)
2. В состав архива с курсововй работой входит пояснительная записка в электронном виде (пример названия:ИКСиС_ПЗ_ВасилийПП_БобриковИП.docx) и архив с исходным кодом проекта(пример названия: ИКСиС_ПО_ВасилийПП_ БобриковИП.zip)
3. Сделать пулреквест в основную ветку на внесенные изменения
